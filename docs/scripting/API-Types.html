is_object(val)​➜

Returns whether a value is a struct, but not a wrapper for a data structure/etc.

is_builtin(val)​➜

Returns whether a value is a wrapper for a data structure/etc. and not a struct.

is_double(val)​➜

In "legacy" NTGML, is_real is an alias for is_numeric and this is the actual is_real.

And the following are extra:

string_is_real(str)​➜

Returns whether a string contains a valid number (signed, fractional, no exponents/infinities).

string_to_real(str, default_value = 0)​➜

Converts a string to a number if it's valid (rules per above) or returns default_value if it isn't.

In "legacy" NTGML, real(str) is equivalent to string_to_real(str, 0) for backwards compatibility.
